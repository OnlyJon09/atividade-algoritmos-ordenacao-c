# Implementação de Algoritmos de Ordenação em C

Projeto desenvolvido para implementar e comparar algoritmos de ordenação utilizando linguagem C.

## Algoritmos implementados
- Bubble Sort
- Selection Sort
- Insertion Sort
- Quick Sort
- Merge Sort
- Shell Sort

## Funcionalidades
- Menu interativo
- Ordenação de vetores de diferentes tamanhos
- Medição de tempo de execução
- Impressão do vetor antes e depois
